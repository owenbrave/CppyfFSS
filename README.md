# 项目简介

## 核心功能
- 基于YOLOv8的银杏叶病检测系统，支持识别chlorosis、insect_pest、physical_damage三类病害
- 提供图片、视频及摄像头三种检测方式
- 支持自定义置信度和IOU阈值
- 可选择不同检测模型
- 界面实时显示目标位置、目标总数、置信度等信息
- 支持批量检测并查看所有结果
- 支持检测结果的保存功能
- 提供完整源码、训练好的模型、4000张标注数据集及运行教程

## 适用场景
- 农业植物病害检测与研究
- 银杏种植健康管理
- 农作物病虫害监测
- 农业科研数据分析
- 植物保护工作辅助

## 技术特点
- 采用Pycharm+Anaconda或VSCode+Anaconda开发环境
- 基于Python 3.9，集成opencv-python、PyQt5、ultralytics、torch1.9等技术栈
- 提供图形用户界面，操作简便
- 模型已预训练，配置环境后可直接使用
- 支持多种输入源检测，灵活适用不同场景
